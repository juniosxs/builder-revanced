# Builder Revanced

Simple Script to Build ReVanced using revanced-cli and Java with Termux.

## How to Use

paste below command, when done go to builder-revanced directory and run Youtube or YouTube Music script, APK to be patched needs to be in "/sdcard/Download" directory and patched apk with revanced you can find in directory "/sdcard".

```
curl -sLo builder-revanced-termux-setup.sh https://github.com/juniosxs/builder-revanced/raw/main/termux-setup.sh
chmod +x builder-revanced-termux-setup.sh
./builder-revanced-termux-setup.sh
cd builder-revanced
```
